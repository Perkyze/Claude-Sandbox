# Claude Code Configuration

This directory contains Claude Code configuration for this project.

## Structure

- `settings.json` - Project-specific settings for Claude Code
- `commands/` - Custom slash commands you can use in Claude Code

## Custom Commands

You can create custom slash commands by adding `.md` files to the `commands/` directory.

Current commands:
- `/project-info` - Get an overview of the project
- `/test` - Run all tests in the project

## Usage

To use a custom command, simply type `/command-name` in Claude Code.

Visit https://docs.claude.com/en/docs/claude-code for more information.
